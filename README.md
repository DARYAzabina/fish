# fish
**Автор:** Дарья Жабина

## описание
компактное приложение для управления ежедневными задачами. позволяет как выбирать случайные активности из базы, так и добавлять собственные

##  инструкция
1. запустите `task_gen.py`
2. выберите категорию или нажмите "создать" для случайного выбора.
3. используйте фильтр над списком истории,чтобы увидеть задачи только по учебе , спорту или работе.
4. ваши данные автоматические сохраняются в файл history.json.
